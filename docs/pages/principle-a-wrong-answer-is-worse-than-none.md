title:: A wrong answer is worse than no answer
type:: principle

- An answer we cannot trust is refused outright — recorded as nothing, never as something plausible.
- Where it already operates, verbatim:
  - [[query-text]] — a country never qualifies a name, and "if the country is all it has, it emits nothing at all — asking a question we cannot trust the answer to is worse than not asking."
  - [[open-when-two-posts-say-the-same-thing]] — the missing-person trap: near-identical wording meaning opposite things, where "a rule that quietly merges them is worse than no rule at all."
  - [[coverage]] — top mode results are "never counted as complete"; coverage is claimed only from latest, the one mode whose completeness we can defend.
