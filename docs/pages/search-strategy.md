title:: Search Strategy
type:: hub
revised:: 2026-07-23

- Doctrine for collecting: how we decide what to ask, over which windows, and in which order. Doctrine changes as we learn; `revised::` says when it last did.
- Seeded from what the record already shows — to be grown here as the thinking happens:
  - Base sweeps first: every real Search so far was a base sweep, and Terms narrow a Search only when a sweep proves too broad — see [[decision-2026-07-22-term-off-switch]] and [[search-term]].
  - Coverage is counted only from latest mode; top is for sampling and never for coverage — see [[coverage]] and [[open-latest-vs-top]].
  - Extend, never recreate: a Search grows one dimension at a time and nothing already collected is put in doubt — see [[search]].
  - Collecting on proposed Places is how geography gets tested: we do not settle whether a Place is real before looking — see [[place]].
