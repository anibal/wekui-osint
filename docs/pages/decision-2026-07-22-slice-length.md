title:: The default slice length is ten minutes
type:: decision

- **Decision** — when a [[search]] does not set a slice length, it is ten minutes.
- Worth remembering what actually ran, though: the twelve real Searches used one hour, two hours, four hours, one day and one week.
- Ten minutes is six times finer than the finest slice ever used, so it produces roughly six times as many Queries for the same window.
- Every Search may set its own, and this is only what happens when none is given.
