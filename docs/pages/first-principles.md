title:: First Principles
type:: hub

- The values this system obeys. None was invented for this page: each was mined from the rules and decisions where it already operates, quotes its sources verbatim, and links to every place it bites. This is the slowest-changing layer of the documentation.
- [[principle-evidence-over-impression]] — claims about how well something works are answered with measurements, not opinions.
- [[principle-unknown-is-never-zero]] — an unknown is recorded as unknown; a blank is knowledge about us, a zero is a claim about the world.
- [[principle-never-rewrite-the-record]] — what happened, happened: append, never edit, never delete.
- [[principle-rather-ask-twice-than-lose-a-name]] — when completeness and economy conflict, completeness wins.
- [[principle-a-wrong-answer-is-worse-than-none]] — an answer we cannot trust is refused outright, not recorded.
- [[principle-one-name-per-fact]] — every fact lives in exactly one place; a copy will eventually disagree with it.
